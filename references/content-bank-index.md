## Using This Index

When you need post ideas:
1. Pick a topic that fits the day's content mix (educational, personal, or product)
2. Check the suggested format (text vs carousel)
3. Always create original content. These are starting points, not templates.
4. Rotate topics across the week. Never post about the same sub-topic twice in a row.
